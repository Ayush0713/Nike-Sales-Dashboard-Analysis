# Nike-Sales-Dashboard-Analysis (Using MS Excel and Power BI)

# Project Objective
The goal of this project is to transform raw sales data into meaningful insights that help stakeholders quickly understand overall performance and identify growth opportunities.

## Dataset Used
- <a href="https://github.com/Ayush0713/Nike-Sales-Dashboard-Analysis/blob/main/Nike%20Sales%20Dashboard.pbix">Dataset </a>

## Question(KPIs)
- How does the monthly revenue trend change throughout the year, and which months perform the best or worst.
- Which product line generates the highest revenue, and which one is the most profitable.
- Who are the top 10 best-selling products, and what percentage do they contribute to total revenue.
- How is revenue distributed across regions, and which region performs the best.
- Which sales channel (Retail or Online) contributes more to revenue.
- Which gender category (Men, Women, Kids) drives the most sales.
